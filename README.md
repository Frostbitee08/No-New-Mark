NoNewMark
===========

NoNewMark disables the "Recently Updated" mark in iOS. **Note:** You *must* have a jailbroken device to use this tweak.

NoNewMark is [availble on cydia](http://modmyi.com/cydia/com.frostbitee08.nonewmark) for free under the [ModMyI](http://www.modmyi.com) repo.
